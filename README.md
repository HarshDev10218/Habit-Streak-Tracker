# 📈 HabitLoop – Advanced Habit Tracker

HabitLoop is a modern and responsive **Habit Tracking Web Application** that helps users build consistency by tracking daily habits, monitoring streaks, analyzing progress, and maintaining long-term productivity.

The application features an intuitive dashboard, interactive 30-day progress matrix, analytics, dark mode, notifications, and data backup/restore functionality—all built using **HTML**, **CSS**, and **Vanilla JavaScript**. :contentReference[oaicite:0]{index=0} :contentReference[oaicite:1]{index=1}

---

## ✨ Features

- ✅ Create and manage unlimited habits
- 📂 Organize habits into categories
  - Health & Fitness
  - Mindset & Soul
  - Work & Finance
  - Daily Routine
- 🔍 Search habits instantly
- 🎯 Filter habits by:
  - All
  - Pending
  - Completed
- 📅 Interactive 30-day activity tracker
- 🔥 Current & longest streak calculation
- 📊 Weekly completion analytics chart
- 🌙 Dark / Light theme
- 🔔 Browser notification reminders
- 💾 Export habit data as JSON
- 📂 Restore habits from JSON backup
- ✏️ Edit existing habits
- 🗑️ Delete habits with confirmation dialog
- 🔔 Toast notifications
- 📱 Fully responsive design

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/HabitLoop.git
```

Navigate into the project folder:

```bash
cd HabitLoop
```

Open **index.html** in your browser

or run the project using **VS Code Live Server**.

---

## 🖥️ Usage

1. Add a new habit.
2. Select its category.
3. Track your daily progress using the 30-day calendar.
4. Monitor your:
   - Current streak
   - Longest streak
   - Completion rate
5. Search or filter habits.
6. Edit or delete habits whenever needed.
7. Enable browser notifications.
8. Export your data or restore it from a backup.
